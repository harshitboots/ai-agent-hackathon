🚀 AI Hackathon — Step-by-Step Guide
🎯 Follow These Steps Carefully
🔹 STEP 1 — Clone the Repository

Open terminal (VS Code / Codespaces):

git clone <REPO_URL>
cd ai-agent-hackathon
🔹 STEP 2 — Install Requirements
pip install -r requirements.txt
🔹 STEP 3 — Run Notebooks (IMPORTANT)

Go to notebooks/ and run in order:

📘 1. Data Exploration

👉 Understand dataset and choose target

📘 2. Feature Engineering

👉 Clean data and create features

📘 3. Model Training

👉 Train model and save it

📘 4. Generate Predictions

👉 Create final output file

🔹 STEP 4 — Save Your Submission

Your file MUST be saved as:

outputs/YOURNAME_predictions.csv
📌 Format must be:
actual,prediction
🔹 STEP 5 — Build Your AI Agent

Run:

streamlit run app.py
🎯 Your agent should:

Take input

Predict using your model

Show results

🔹 STEP 6 — Test Your Agent

👉 Open browser:

http://localhost:8501

👉 Try different inputs and verify results

🔹 STEP 7 — Final Submission

Before submitting:

✔ Predictions file saved
✔ Agent working
✔ Model saved

🔹 STEP 8 — Push Your Code (Git)
Pull latest changes:
git pull origin main
Add your work:
git add output/result.csv
Commit:
git commit -m "Final submission - YOUR NAME"
Push:
git push origin main
🔹 STEP 9 — Create Pull Request

On GitHub:

Click Compare & Pull Request

Click Create Pull Request
